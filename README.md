# Todo List

Todo list web app built in Django.


## Installation

Install required packages with pip

```bash
  pip install django
```

<b>To Run</b>

1. Make Migrations
```bash
  python manage.py makemigrations
```

2. Migrate to database
```bash
  python manage.py makemigrations
```

3. Create a super user
```bash
  python manage.py createsuperuser
```

4. Run Server
```bash
  python manage.py runserver
```
